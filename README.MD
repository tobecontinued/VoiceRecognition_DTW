#VoiceRecognition_DTW
这是一个基于MFCC和DTW的语音孤词识别程序，应该算一种比较基本的语音识别程序
##使用
###Linux:
在VoiceRecognition目录下执行：<br>
`make && ./VoiceRecognition_DTW`<br>
###Windows:
使用VS2010以上打开工程即可
